# Self-Attention

Self-attention, also known as intra-attention, is an attention mechanism relating different positions of a single sequence in order to compute a representation of the same sequence.

## 🖼️ Diagram
![Self-Attention Diagram](../images/self_attention.png)

## 📄 Original Paper
- **Title:** [Long Short-Term Memory-Networks for Machine Reading](https://arxiv.org/abs/1601.06733)
- **Year:** 2016
- **Authors:** Cheng et al.

## 💡 Key Concept
In self-attention, the Query, Key, and Value vectors all come from the same input sequence. This allows the model to learn dependencies between different words in the same sentence, regardless of their distance.
